# 🍔 Smart Online Food Ordering System With AI 🚀

A **full-stack MERN-based online food ordering platform** enhanced with **AI-powered features**, real-time updates, and multi-role architecture (Customer, Seller, Admin).

This project simulates a real-world system similar to modern platforms and includes **intelligent recommendations, chatbot integration, and scalable backend architecture**.

- Customers ordering food
- Sellers managing shops and items
- Admin controlling the entire platform
- AI enhancing user experience

# ⚙️ Tech Stack

## 🔹 Frontend
- React.js
- Tailwind CSS
- Zustand / Redux
- Framer Motion

## 🔹 Backend
- Node.js
- Express.js
- MongoDB (Mongoose)
